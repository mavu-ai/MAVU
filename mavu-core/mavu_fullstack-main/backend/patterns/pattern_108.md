# Паттерн 108 — Музыка для настроения

**Цель:**
Использовать музыку как инструмент регуляции эмоционального состояния.

**Принцип:**
Музыка напрямую влияет на эмоции. Энергичная музыка поднимает настроение, спокойная — успокаивает. Ребёнок может научиться выбирать музыку в зависимости от своего состояния.

**Подробное описание:**
Обсудить с ребёнком, какая музыка вызывает какие чувства. Создать плейлисты для разных ситуаций: для успокоения, для радости, для концентрации, для выплеска энергии.

**Примеры фраз:**
- «Какую музыку ты хочешь послушать сейчас?»
- «Когда грустно — иногда помогает грустная музыка, а иногда — весёлая. Что ты хочешь?»
- «Давай сделаем плейлист для спокойствия. Какие песни тебя успокаивают?»
- «Потанцуем под музыку? Это помогает выпустить энергию.»

**Что избегать:**
- Не навязывать свой вкус.
- Не использовать громкую музыку перед сном.
- Не запрещать грустную музыку — она тоже нужна.

**Дополнительно:**
- Совместное слушание музыки — способ сблизиться.
- Классическая музыка часто успокаивает.
- Создание музыки (барабанить, петь) — выход для эмоций.

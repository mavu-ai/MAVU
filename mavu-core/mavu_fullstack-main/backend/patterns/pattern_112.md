# Паттерн 112 — Создание «коробки спокойствия»

**Цель:**
Подготовить набор предметов, которые помогают ребёнку успокоиться в стрессовых ситуациях.

**Принцип:**
Сенсорные объекты (мягкие, приятные на ощупь, ароматные) помогают заземлиться. Когда они собраны заранее, ребёнок может использовать их самостоятельно.

**Подробное описание:**
Вместе с ребёнком создать коробку с предметами для успокоения: антистресс-игрушка, мягкая ткань, фото близких, камушек, карточки с дыхательными упражнениями, любимая маленькая игрушка. Хранить в доступном месте.

**Примеры фраз:**
- «Давай соберём коробку, которая будет помогать тебе успокоиться.»
- «Что бы ты хотел туда положить? Что приятно трогать, на что приятно смотреть?»
- «Когда чувствуешь тревогу — открой коробку и выбери что-нибудь.»
- «Это твоя личная коробка спокойствия.»

**Что избегать:**
- Не класть туда хрупкие или опасные предметы.
- Не использовать как наказание («раз плохо себя вёл — коробку заберу»).
- Не забирать без разрешения.

**Дополнительно:**
- Можно сделать мини-версию для школы или сумки.
- Обновлять содержимое по мере взросления.
- Научить использовать самостоятельно.

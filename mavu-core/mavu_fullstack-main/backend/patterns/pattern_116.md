# Паттерн 116 — Карточки эмоций

**Цель:**
Использовать визуальные карточки для помощи ребёнку в определении и выражении эмоций.

**Принцип:**
Детям, особенно маленьким, легче указать на картинку, чем подобрать слово. Карточки с лицами, выражающими разные эмоции, помогают в коммуникации.

**Подробное описание:**
Набор карточек с изображением эмоций: радость, грусть, злость, страх, удивление, стыд и т.д. Ребёнок выбирает карточку, которая соответствует его состоянию. Можно обсуждать, что вызвало эту эмоцию.

**Примеры фраз:**
- «Покажи мне, какая карточка похожа на то, что ты сейчас чувствуешь.»
- «Ты выбрал "грусть". Расскажи, что случилось?»
- «А какой бы ты хотел, чтобы было твоё настроение?»
- «Давай выберем карточку для этого персонажа из книги.»

**Что избегать:**
- Не ограничивать выбор одной карточкой — можно несколько.
- Не говорить «неправильно выбрал».
- Не использовать только для негативных эмоций.

**Дополнительно:**
- Можно сделать карточки вместе (нарисовать).
- Использовать для обсуждения историй и фильмов.
- Носить несколько карточек с собой.

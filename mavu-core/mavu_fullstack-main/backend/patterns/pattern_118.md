# Паттерн 118 — Позитивное самовнушение (аффирмации для детей)

**Цель:**
Научить ребёнка использовать позитивные утверждения для повышения уверенности и снижения тревоги.

**Принцип:**
Мысли влияют на чувства. Регулярное повторение позитивных утверждений помогает формировать более здоровое отношение к себе. Важно, чтобы утверждения были реалистичными.

**Подробное описание:**
Создать вместе с ребёнком несколько коротких позитивных фраз, которые он может говорить себе. Повторять утром, перед сложными ситуациями, перед сном.

**Примеры фраз:**
- «Я могу справиться с трудностями.»
- «Я достаточно хорош, даже если не всё получается.»
- «Ошибки помогают мне учиться.»
- «У меня есть люди, которые меня любят.»
- «Я становлюсь сильнее каждый день.»

**Что избегать:**
- Не использовать нереалистичные утверждения («я лучший во всём»).
- Не заставлять повторять без понимания смысла.
- Не использовать как замену реальной поддержке.

**Дополнительно:**
- Написать аффирмации на стикерах и развесить.
- Говорить вместе, глядя в зеркало.
- Позволить ребёнку создать свои собственные фразы.

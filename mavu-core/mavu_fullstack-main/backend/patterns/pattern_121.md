# Паттерн 121 — Естественные последствия вместо наказаний

**Цель:**
Позволить ребёнку учиться на естественных последствиях своих действий вместо искусственных наказаний.

**Принцип:**
Естественные последствия — результат самого действия, а не решение взрослого. Они учат причинно-следственным связям: не надел шапку — замёрз, не убрал игрушки — не нашёл.

**Подробное описание:**
Позволить последствиям наступить (если они безопасны). Не злорадствовать: «Я же говорил!» Использовать момент для обучения: «Что можно сделать по-другому в следующий раз?»

**Примеры фраз:**
- «Ты решил не взять зонт, и вот промок. В следующий раз как поступишь?»
- «Игрушка сломалась, потому что бросил её. Это обидно. Можешь попробовать починить.»
- «Не успел сделать домашку — придётся объяснить учителю.»
- «Что ты понял из этой ситуации?»

**Что избегать:**
- Не использовать, если последствия опасны (не дать обжечься об огонь).
- Не злорадствовать и не говорить «я же предупреждал».
- Не добавлять свои наказания сверху.

**Дополнительно:**
- Заранее предупредить о возможных последствиях (один раз).
- После — обсудить, не читать нотации.
- Этот метод развивает ответственность.

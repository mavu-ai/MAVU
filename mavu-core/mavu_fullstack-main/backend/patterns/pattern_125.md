# Паттерн 125 — Время один на один

**Цель:**
Регулярно проводить качественное время с ребёнком для укрепления связи и эмоциональной близости.

**Принцип:**
Качество важнее количества. Даже 15 минут полного внимания ценнее, чем часы рядом, но с телефоном. Это время, когда ребёнок чувствует себя единственным и самым важным.

**Подробное описание:**
Выделить регулярное время (ежедневно или несколько раз в неделю), когда взрослый полностью сосредоточен на ребёнке. Без телефона, без других детей, без домашних дел. Занятие выбирает ребёнок.

**Примеры фраз:**
- «Сейчас наше особенное время. Чем хочешь заняться?»
- «Эти 20 минут только для тебя. Я весь внимание.»
- «Я отложу телефон — хочу быть с тобой полностью.»
- «Мне нравится проводить время с тобой.»

**Что избегать:**
- Не отменять без важной причины.
- Не использовать для обучения или нотаций.
- Не смотреть в телефон.

**Дополнительно:**
- Можно дать этому времени название: «Время с мамой/папой».
- Для нескольких детей — отдельное время с каждым.
- Даже тихое время вместе (лежать рядом, смотреть в окно) — ценно.

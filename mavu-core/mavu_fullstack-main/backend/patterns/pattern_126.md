# Паттерн 126 — Переходный объект (игрушка-друг)

**Цель:**
Использовать любимую игрушку как источник безопасности и комфорта для ребёнка.

**Принцип:**
Переходный объект (по Д. Винникотту) — это предмет, который символизирует связь с родителем и безопасность. Он помогает ребёнку справляться с разлукой и тревогой.

**Подробное описание:**
Любимая игрушка, одеяло или мягкий предмет, который ребёнок берёт с собой. Не отбирать и не высмеивать эту привязанность. Разрешать брать в сложные ситуации.

**Примеры фраз:**
- «Хочешь взять мишку с собой? Он будет рядом.»
- «Твой зайка может помочь тебе чувствовать себя смелее.»
- «Обними игрушку, когда станет страшно.»
- «Мишка будет напоминать тебе, что мама тебя любит.»

**Что избегать:**
- Не отбирать и не терять специально.
- Не высмеивать привязанность.
- Не использовать как инструмент манипуляции.

**Дополнительно:**
- Иметь «дублёр» любимой игрушки на случай потери.
- С возрастом привязанность естественно ослабевает.
- Это нормально и для школьников.

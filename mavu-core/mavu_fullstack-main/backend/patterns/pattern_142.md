# Паттерн 142 — Отказ есть

**Цель:**
Реагировать на отказ от еды без давления, сохраняя здоровые отношения с пищей.

**Принцип:**
Принуждение к еде создаёт негативные ассоциации и проблемы с питанием в будущем. Ребёнок сам регулирует, сколько есть. Задача родителя — предложить здоровую еду.

**Подробное описание:**
Не заставлять есть. Не уговаривать, не шантажировать десертом. Предложить еду, дать время, убрать без комментариев. Следующий приём пищи — по расписанию.

**Примеры фраз:**
- «Ты не голоден? Хорошо. Следующий приём пищи — через N часов.»
- «Попробуй кусочек. Не нравится — можешь не есть.»
- «Твоё тело знает, сколько ему нужно.»
- «Я не буду заставлять. Но перекуса до обеда не будет.»

**Что избегать:**
- Не кормить насильно.
- Не использовать еду как награду/наказание.
- Не делать отдельные блюда «только для ребёнка».

**Дополнительно:**
- Есть вместе за столом — пример.
- Маленькие порции снижают давление.
- Привлекать к готовке — повышает интерес к еде.

# Паттерн 143 — Не хочет делиться

**Цель:**
Научить ребёнка делиться без принуждения и стыда.

**Принцип:**
Делиться — социальный навык, который развивается постепенно. Принуждение не учит щедрости, а вызывает сопротивление. Ребёнок должен чувствовать контроль над своими вещами.

**Подробное описание:**
Не заставлять отдавать свои вещи. Учить по очереди. Объяснять чувства другого. Хвалить, когда делится добровольно. Уважать право на «только мои» вещи.

**Примеры фраз:**
- «Это твоя игрушка, и ты решаешь, давать ли её.»
- «Смотри, Маша грустит, что не может поиграть. Хочешь дать ей на минутку?»
- «Можно играть по очереди: сначала ты, потом он.»
- «Ты поделился! Смотри, как он обрадовался.»

**Что избегать:**
- Не отбирать силой для другого ребёнка.
- Не стыдить: «Ты жадина».
- Не требовать делиться всем.

**Дополнительно:**
- До 3 лет делиться очень сложно — это нормально.
- Особо ценные игрушки можно убрать до прихода гостей.
- Моделировать делёж между взрослыми.

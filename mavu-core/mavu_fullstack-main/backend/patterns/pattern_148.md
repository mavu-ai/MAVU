# Паттерн 148 — Интернет и экранное время

**Цель:**
Установить здоровые границы использования гаджетов без постоянных конфликтов.

**Принцип:**
Экраны — часть современной жизни. Запретить полностью нереалистично. Нужны чёткие, согласованные правила и альтернативные занятия.

**Подробное описание:**
Установить конкретные правила: сколько времени, когда, какой контент. Использовать таймер. Договориться о последствиях нарушения. Предложить другие интересные занятия.

**Примеры фраз:**
- «У нас правило: 1 час экранного времени в день. Как хочешь его использовать?»
- «Таймер прозвенит через 30 минут — и мы выключаем.»
- «После экрана — прогулка/игра/чтение.»
- «Что интересного можно сделать без экрана?»

**Что избегать:**
- Не использовать экран как «затычку».
- Не менять правила каждый день.
- Не забирать экран в гневе без предупреждения.

**Дополнительно:**
- Совместное использование (смотреть вместе, обсуждать).
- Экран не ближе чем за час до сна.
- Быть примером в использовании гаджетов.

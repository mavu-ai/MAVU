# Паттерн 165 — Вежливость и хорошие манеры

**Цель:**
Обучить ребёнка базовым правилам вежливости в повседневной жизни.

**Принцип:**
Вежливость — социальный клей. «Пожалуйста», «спасибо», «извините» — не пустые формальности, а проявление уважения к другим. Учат примером, а не требованием.

**Подробное описание:**
Моделировать вежливое поведение. Напоминать мягко, не требовать публично. Объяснять, почему вежливость важна. Хвалить за вежливое поведение.

**Примеры фраз:**
- «Когда просишь — говорим "пожалуйста". Когда получаешь — "спасибо".»
- «Вежливость — это способ показать уважение.»
- «Людям приятно, когда с ними вежливы.»
- «Ты так вежливо попросил — мне приятно!»

**Что избегать:**
- Не требовать вежливости в унизительной форме.
- Не заставлять публично: «Что нужно сказать?!»
- Не быть невежливым самому.

**Дополнительно:**
- Ролевые игры с «волшебными словами».
- Личный пример — главный учитель.
- Вежливость к ребёнку учит вежливости.

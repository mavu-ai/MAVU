# Паттерн 193 — Незнакомые люди и безопасность

**Цель:**
Научить ребёнка правилам безопасности с незнакомцами без создания паранойи.

**Принцип:**
Не все незнакомцы опасны, но есть правила безопасности. Важно научить конкретным правилам, а не создавать страх всех людей.

**Подробное описание:**
Объяснить конкретные правила: не уходить с незнакомцами, не брать от них подарки без разрешения родителей, звать на помощь, знать свои данные (имя, телефон родителей).

**Примеры фраз:**
- «Незнакомец — тот, кого ты не знаешь. Не все они плохие, но есть правила.»
- «Никогда не уходи с незнакомцем, даже если он говорит, что мама попросила.»
- «Если кто-то пугает — кричи и беги к другим людям.»
- «Наш семейный пароль — [слово]. Только тот, кто его знает, может тебя забрать.»

**Что избегать:**
- Не пугать историями о похищениях.
- Не создавать страх всех взрослых.
- Не игнорировать тему.

**Дополнительно:**
- Семейный пароль для проверки.
- Отработать ситуации: что делать, если потерялся.
- Обсуждать безопасных взрослых (полиция, продавец).

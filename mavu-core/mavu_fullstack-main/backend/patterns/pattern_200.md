# Паттерн 200 — Принятие решений

**Цель:**
Учить ребёнка принимать решения самостоятельно, развивая этот навык постепенно.

**Принцип:**
Умение принимать решения — ключевой жизненный навык. Он развивается через практику: сначала маленькие решения, потом более значимые.

**Подробное описание:**
Давать выбор по возрасту. Обсуждать процесс принятия решений: плюсы и минусы вариантов. Не исправлять «неправильный» выбор, если он безопасен. Анализировать результаты вместе.

**Примеры фраз:**
- «Это твой выбор. Подумай и реши.»
- «Какие есть варианты? Что хорошего и плохого в каждом?»
- «Ты принял решение. Посмотрим, что получится.»
- «Как думаешь, это решение было хорошим? Почему?»

**Что избегать:**
- Не решать всё за ребёнка.
- Не критиковать «неправильный» выбор.
- Не перегружать важными решениями.

**Дополнительно:**
- Начинать с маленьких решений (что надеть, что есть на завтрак).
- Постепенно увеличивать значимость.
- Учить жить с последствиями решений.

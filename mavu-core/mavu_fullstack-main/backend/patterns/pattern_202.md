# Паттерн 202 — Постановка целей

**Цель:**
Научить ребёнка ставить достижимые цели и работать над ними.

**Принцип:**
Умение ставить цели — важный навык. Цели должны быть конкретными, достижимыми и значимыми для ребёнка.

**Подробное описание:**
Помогать формулировать цели. Разбивать большие цели на шаги. Отмечать прогресс. Праздновать достижения.

**Примеры фраз:**
- «Чего ты хочешь добиться? Давай запишем.»
- «Это большая цель. Какой первый шаг?»
- «Смотри, ты уже сделал это и это — ты движешься к цели!»
- «Что тебе нужно, чтобы достичь этого?»

**Что избегать:**
- Не ставить цели за ребёнка.
- Не делать цели слишком сложными.
- Не забывать отмечать шаги.

**Дополнительно:**
- Визуальный трекер прогресса.
- Регулярный пересмотр целей.
- Маленькие цели для начала.

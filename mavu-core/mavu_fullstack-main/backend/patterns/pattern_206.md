# Паттерн 206 — Настойчивость

**Цель:**
Развивать настойчивость и умение продолжать, когда трудно.

**Принцип:**
Настойчивость — ключевое качество для достижения целей. Важно учить не сдаваться при первых трудностях.

**Подробное описание:**
Признавать трудности. Напоминать о прошлых преодолениях. Поддерживать в моменты сомнений. Отмечать усилия.

**Примеры фраз:**
- «Сложно, но ты справишься. Ты уже справлялся с трудным.»
- «Помнишь, как ты научился [навык]? Тоже было непросто.»
- «Настойчивость — это суперсила.»
- «Ещё немного — и получится.»

**Что избегать:**
- Не говорить «это легко».
- Не сравнивать с другими.
- Не оставлять одного с трудностью.

**Дополнительно:**
- Истории о настойчивых людях.
- Праздновать преодоление.
- Помощь в сложные моменты.

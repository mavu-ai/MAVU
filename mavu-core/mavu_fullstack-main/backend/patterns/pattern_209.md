# Паттерн 209 — Концентрация внимания

**Цель:**
Помочь ребёнку развить способность концентрироваться.

**Принцип:**
Концентрация — навык, который развивается. Важно создавать условия и постепенно увеличивать время сосредоточения.

**Подробное описание:**
Убирать отвлекающие факторы. Делать перерывы. Использовать таймер. Начинать с коротких периодов.

**Примеры фраз:**
- «Давай уберём всё лишнее и сосредоточимся.»
- «5 минут полной концентрации — потом перерыв.»
- «Когда фокусируешься, мозг работает лучше.»
- «Одно дело за раз — так эффективнее.»

**Что избегать:**
- Не требовать долгой концентрации сразу.
- Не критиковать за отвлечение.
- Не перегружать задачами.

**Дополнительно:**
- Техника помодоро для детей.
- Игры на внимание.
- Тихое рабочее место.

# Паттерн 210 — Планирование

**Цель:**
Научить ребёнка планировать свои действия и время.

**Принцип:**
Планирование — важный навык для успеха. Важно учить делать это постепенно, начиная с простого.

**Подробное описание:**
Составлять планы вместе. Использовать визуальные планы. Проверять выполнение. Корректировать планы.

**Примеры фраз:**
- «Что нужно сделать сегодня? Давай составим план.»
- «Что сделаем первым? А потом?»
- «План помогает ничего не забыть.»
- «Если план не сработал — его можно изменить.»

**Что избегать:**
- Не делать планы слишком сложными.
- Не ругать за невыполнение.
- Не планировать за ребёнка.

**Дополнительно:**
- Доска с расписанием.
- Планирование выходных вместе.
- Чек-листы для ребёнка.

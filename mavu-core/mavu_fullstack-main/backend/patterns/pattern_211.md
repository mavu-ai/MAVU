# Паттерн 211 — Самодисциплина

**Цель:**
Развивать самодисциплину — умение делать нужное, даже когда не хочется.

**Принцип:**
Самодисциплина — навык, который развивается через практику. Важно начинать с малого и постепенно усложнять.

**Подробное описание:**
Устанавливать правила вместе. Обсуждать последствия. Поддерживать в трудные моменты. Отмечать успехи.

**Примеры фраз:**
- «Сделаем сначала нужное, потом приятное.»
- «Самодисциплина — это когда сам себе хозяин.»
- «Знаю, что не хочется, но ты справишься.»
- «Каждый раз, когда делаешь нужное — становишься сильнее.»

**Что избегать:**
- Не требовать идеальной дисциплины сразу.
- Не полагаться только на наказания.
- Не перегружать правилами.

**Дополнительно:**
- Техника «сначала работа, потом отдых».
- Чёткие, понятные правила.
- Пример взрослых.

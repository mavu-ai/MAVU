# Паттерн 212 — Креативность

**Цель:**
Развивать креативное мышление и творческий подход.

**Принцип:**
Креативность — не врождённый талант, а навык. Важно создавать условия для творческого мышления и не бояться нестандартных идей.

**Подробное описание:**
Задавать открытые вопросы. Поощрять необычные идеи. Не критиковать творческие решения. Давать материалы для творчества.

**Примеры фраз:**
- «А что если сделать по-другому?»
- «Какие ещё идеи приходят в голову?»
- «Интересная мысль! Расскажи подробнее.»
- «Нет неправильных идей в творчестве.»

**Что избегать:**
- Не критиковать необычные идеи.
- Не давать готовые решения.
- Не оценивать творчество как «правильное/неправильное».

**Дополнительно:**
- Свободное рисование.
- Игры на воображение.
- Открытые материалы (конструктор, пластилин).

# Паттерн 216 — Музыкальное развитие

**Цель:**
Поддерживать интерес к музыке и музыкальное развитие.

**Принцип:**
Музыка развивает мозг, эмоциональный интеллект, координацию. Важно знакомить с разной музыкой без принуждения.

**Подробное описание:**
Слушать музыку вместе. Обсуждать, что нравится. Предлагать попробовать инструменты. Не заставлять заниматься.

**Примеры фраз:**
- «Какая музыка тебе нравится?»
- «Послушай, какой интересный звук!»
- «Хочешь попробовать поиграть?»
- «Музыка может выражать разные чувства.»

**Что избегать:**
- Не заставлять заниматься музыкой.
- Не критиковать музыкальные вкусы.
- Не требовать быстрых результатов.

**Дополнительно:**
- Разная музыка: классика, народная, современная.
- Музыкальные игры.
- Концерты и живая музыка.

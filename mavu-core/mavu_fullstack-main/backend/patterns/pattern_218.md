# Паттерн 218 — Изучение языков

**Цель:**
Поддерживать интерес к изучению языков.

**Принцип:**
Языки расширяют мир и развивают мозг. Важно сделать изучение интересным и естественным.

**Подробное описание:**
Показывать ценность языков. Использовать игровые методы. Смотреть мультфильмы на других языках. Не требовать идеальности.

**Примеры фраз:**
- «Когда знаешь язык — можешь говорить с людьми из других стран!»
- «Давай выучим как это по-английски?»
- «Ошибаться при изучении языка — нормально.»
- «Ты уже знаешь столько слов!»

**Что избегать:**
- Не делать изучение скучным.
- Не требовать идеального произношения.
- Не сравнивать с другими.

**Дополнительно:**
- Приложения для изучения языков.
- Песни и мультфильмы.
- Общение с носителями.

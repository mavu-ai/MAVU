# Паттерн 220 — Технологии и гаджеты

**Цель:**
Сформировать здоровое отношение к технологиям.

**Принцип:**
Технологии — инструмент. Важно научить использовать их с пользой и соблюдать баланс.

**Подробное описание:**
Устанавливать правила использования. Обсуждать пользу и риски. Показывать разные способы применения. Быть примером.

**Примеры фраз:**
- «Гаджеты — это инструмент. Как молоток: можно строить, можно ломать.»
- «Давай договоримся о времени использования.»
- «Что полезного можно сделать с помощью планшета?»
- «После экрана глазам нужен отдых.»

**Что избегать:**
- Не использовать гаджеты как награду/наказание.
- Не давать неограниченный доступ.
- Не игнорировать собственный пример.

**Дополнительно:**
- Совместный выбор контента.
- Развивающие приложения.
- Технологии для творчества.

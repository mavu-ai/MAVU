# Паттерн 221 — Финансовая грамотность

**Цель:**
Заложить основы финансовой грамотности.

**Принцип:**
Понимание денег важно для будущего. Важно учить на практике, по возрасту.

**Подробное описание:**
Объяснять, откуда берутся деньги. Давать карманные деньги. Обсуждать выбор при покупках. Учить откладывать.

**Примеры фраз:**
- «Деньги зарабатывают работой.»
- «Если потратить всё сейчас, на потом не останется.»
- «Что для тебя важнее — это или то? Выбирай.»
- «Можно копить на большую мечту.»

**Что избегать:**
- Не пугать бедностью.
- Не скрывать финансовые реальности.
- Не давать всё по первому требованию.

**Дополнительно:**
- Копилка для целей.
- Простые объяснения.
- Игры про магазин.

# Паттерн 225 — Тайм-менеджмент

**Цель:**
Научить ребёнка управлять своим временем.

**Принцип:**
Управление временем — важный навык для жизни. Важно учить постепенно, с простого.

**Подробное описание:**
Использовать визуальные таймеры. Обсуждать, сколько времени уходит на задачи. Планировать день вместе.

**Примеры фраз:**
- «Сколько времени тебе нужно на это?»
- «Смотри на часы — у тебя ещё 10 минут.»
- «Давай спланируем, что успеем сделать.»
- «Время — ресурс. Его можно тратить с умом.»

**Что избегать:**
- Не торопить постоянно.
- Не перегружать расписание.
- Не делать время врагом.

**Дополнительно:**
- Визуальные таймеры.
- Песочные часы.
- Расписание с картинками.

# Паттерн 229 — Публичные выступления

**Цель:**
Помочь ребёнку преодолеть страх публичных выступлений.

**Принцип:**
Страх выступлений распространён. Важно создать безопасную среду для практики и постепенно увеличивать аудиторию.

**Подробное описание:**
Практиковаться дома. Начинать с маленькой аудитории. Обсуждать, что волнение нормально. Замечать успехи.

**Примеры фраз:**
- «Волноваться перед выступлением — нормально. Даже взрослые волнуются.»
- «Давай потренируемся дома.»
- «Ты отлично выступил! Что понравилось самому?»
- «Чем больше практики — тем легче.»

**Что избегать:**
- Не высмеивать страх.
- Не заставлять выступать неподготовленным.
- Не критиковать публично.

**Дополнительно:**
- Репетиции перед зеркалом.
- Выступления перед семьёй.
- Техники дыхания перед выступлением.

# Паттерн 231 — Гигиена

**Цель:**
Привить навыки личной гигиены.

**Принцип:**
Гигиена важна для здоровья. Важно объяснять зачем, а не просто заставлять.

**Подробное описание:**
Объяснять, зачем нужна гигиена. Создавать рутины. Делать процесс приятным. Хвалить за самостоятельность.

**Примеры фраз:**
- «Микробы любят грязные руки. Помоем — и они уйдут!»
- «Чистые зубы — здоровые зубы.»
- «Давай сделаем это весело!»
- «Ты сам почистил зубы — молодец!»

**Что избегать:**
- Не пугать болезнями чрезмерно.
- Не превращать в наказание.
- Не делать за ребёнка всегда.

**Дополнительно:**
- Яркие принадлежности.
- Песенки для чистки зубов.
- Таблица отслеживания.

# Паттерн 232 — Режим дня

**Цель:**
Создать стабильный режим дня.

**Принцип:**
Режим даёт предсказуемость и безопасность. Регулярность важна для здоровья и эмоционального состояния.

**Подробное описание:**
Устанавливать регулярное время для еды, сна, активностей. Придерживаться режима. Обсуждать расписание с ребёнком.

**Примеры фраз:**
- «Сейчас время для [активность].»
- «Режим помогает телу знать, чего ожидать.»
- «Давай посмотрим, что у нас по расписанию.»
- «Когда есть режим — меньше ссор и больше времени на приятное.»

**Что избегать:**
- Не делать режим тюрьмой.
- Не менять резко.
- Не игнорировать потребности ребёнка.

**Дополнительно:**
- Визуальное расписание.
- Гибкость в выходные.
- Совместное планирование.

# Паттерн 233 — Отдых и восстановление

**Цель:**
Научить ребёнка отдыхать и восстанавливать силы.

**Принцип:**
Отдых важен не меньше, чем активность. Важно учить замечать усталость и давать себе отдых.

**Подробное описание:**
Показывать важность отдыха. Создавать время для отдыха. Обсуждать признаки усталости.

**Примеры фраз:**
- «Ты устал? Давай отдохнём.»
- «Отдых — это не лень, это восстановление.»
- «Что помогает тебе восстановить силы?»
- «После отдыха сил будет больше.»

**Что избегать:**
- Не перегружать расписание.
- Не критиковать за усталость.
- Не называть отдых ленью.

**Дополнительно:**
- Тихое время каждый день.
- Разные виды отдыха.
- Наблюдение за признаками усталости.

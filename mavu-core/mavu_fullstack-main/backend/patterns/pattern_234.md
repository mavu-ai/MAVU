# Паттерн 234 — Хобби и увлечения

**Цель:**
Поддерживать развитие хобби и увлечений.

**Принцип:**
Хобби важны для счастья и развития. Важно поддерживать интересы ребёнка, даже если они отличаются от ваших.

**Подробное описание:**
Замечать интересы. Предоставлять ресурсы. Интересоваться увлечениями. Не оценивать «серьёзность» хобби.

**Примеры фраз:**
- «Расскажи мне про это — тебе явно нравится!»
- «Что тебе нужно для этого хобби?»
- «Классно, что у тебя есть такое увлечение.»
- «Хобби делают жизнь интереснее.»

**Что избегать:**
- Не обесценивать хобби.
- Не навязывать свои увлечения.
- Не требовать результатов.

**Дополнительно:**
- Время для хобби в расписании.
- Материалы и инструменты.
- Интерес к процессу, не результату.

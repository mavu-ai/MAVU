# Паттерн 236 — Перфекционизм

**Цель:**
Помочь ребёнку справляться с перфекционизмом.

**Принцип:**
Перфекционизм может парализовать и приводить к тревоге. Важно учить принимать «достаточно хорошо».

**Подробное описание:**
Признавать стремление к совершенству. Показывать, что ошибки нормальны. Хвалить за усилия, не только результат.

**Примеры фраз:**
- «Не обязательно быть идеальным.»
- «Достаточно хорошо — это тоже хорошо.»
- «Ошибки помогают учиться.»
- «Важнее попробовать, чем сделать идеально.»

**Что избегать:**
- Не требовать идеальности.
- Не сравнивать с «идеальными» детьми.
- Не критиковать за несовершенство.

**Дополнительно:**
- Модель «черновик и чистовик».
- Истории о несовершенных, но успешных людях.
- Принятие своих ошибок вслух.

# Паттерн 23 — Предупреждение заранее

**Цель:**  
Снизить сопротивление ребёнка при смене активности.

**Принцип:**  
Ребёнку заранее сообщается о предстоящих изменениях, чтобы у него было время подготовиться.

**Подробное описание:**  
Внезапные требования («Собирайся, мы уходим сейчас!») вызывают протест. Если заранее предупредить, ребёнку легче переключиться. Это формирует навык планирования и снижает тревожность.

**Примеры фраз:**  
- «Через пять минут мы заканчиваем игру и идём домой.»  
- «Скоро пора ужинать, заканчивай рисунок.»  
- «Через два дня у тебя контрольная, давай начнём готовиться понемногу.»  
- «Через полчаса мы уходим, можешь доиграть.»

**Что избегать:**  
- Не требовать немедленного прекращения без предупреждения.  
- Не обманывать («Через пять минут», а на самом деле через 30 секунд).  
- Не кричать вместо предупреждения.

**Дополнительно:**  
- Для малышей можно использовать визуальные подсказки (песочные часы, таймер).  
- Подросткам полезно обсуждать расписание заранее.

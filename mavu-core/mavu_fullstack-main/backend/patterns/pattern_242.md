# Паттерн 242 — Следование за другими

**Цель:**
Научить быть хорошим членом команды.

**Принцип:**
Не все должны быть лидерами. Умение следовать, сотрудничать, поддерживать — тоже важные навыки.

**Подробное описание:**
Ценить разные роли. Обсуждать важность команды. Не принуждать к лидерству.

**Примеры фраз:**
- «Команде нужны разные люди.»
- «Быть хорошим помощником — тоже важно.»
- «Ты отлично поддержал друга.»
- «Не обязательно всегда быть главным.»

**Что избегать:**
- Не заставлять всегда лидировать.
- Не обесценивать роль помощника.
- Не критиковать за нежелание командовать.

**Дополнительно:**
- Разные роли в играх.
- Обсуждение команд в спорте.
- Ценность каждого вклада.

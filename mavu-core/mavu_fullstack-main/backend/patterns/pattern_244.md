# Паттерн 244 — Адаптивность

**Цель:**
Развивать гибкость и умение адаптироваться к изменениям.

**Принцип:**
Мир меняется. Умение адаптироваться — важный навык. Важно учить принимать изменения.

**Подробное описание:**
Обсуждать изменения заранее. Помогать найти позитив. Показывать, что изменения — это возможности.

**Примеры фраз:**
- «Изменения бывают сложными, но ты справишься.»
- «Что хорошего может быть в этом изменении?»
- «Мы адаптируемся — это наша суперсила.»
- «Не всё идёт по плану — и это нормально.»

**Что избегать:**
- Не игнорировать сложность изменений.
- Не менять резко без подготовки.
- Не критиковать за сопротивление.

**Дополнительно:**
- Маленькие изменения для практики.
- Обсуждение примеров адаптации.
- План Б.

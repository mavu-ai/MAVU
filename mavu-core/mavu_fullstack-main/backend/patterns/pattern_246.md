# Паттерн 246 — Решение проблем

**Цель:**
Развивать навык решения проблем.

**Принцип:**
Умение решать проблемы — ключевой навык для жизни. Важно учить процессу, а не давать готовые решения.

**Подробное описание:**
Задавать вопросы вместо ответов. Обсуждать варианты. Позволять пробовать и ошибаться.

**Примеры фраз:**
- «Что ты можешь сделать в этой ситуации?»
- «Какие есть варианты? Давай подумаем вместе.»
- «Это не сработало — что попробуем дальше?»
- «Каждая проблема имеет решение.»

**Что избегать:**
- Не решать все проблемы за ребёнка.
- Не критиковать за неудачные попытки.
- Не говорить «это просто» когда сложно.

**Дополнительно:**
- Алгоритм решения: проблема-варианты-выбор-действие.
- Обсуждение примеров решения проблем.
- Головоломки и задачки.

# Паттерн 247 — Терпение в обучении

**Цель:**
Развивать терпение в процессе обучения.

**Принцип:**
Обучение требует времени. Важно учить не сдаваться и ценить процесс, не только результат.

**Подробное описание:**
Напоминать о прошлых успехах. Разбивать на этапы. Праздновать маленькие победы.

**Примеры фраз:**
- «Обучение требует времени — и это нормально.»
- «Помнишь, как раньше ты не умел это? А теперь умеешь!»
- «Каждый раз ты становишься чуть лучше.»
- «Терпение — часть мастерства.»

**Что избегать:**
- Не торопить.
- Не сравнивать с другими.
- Не говорить «это легко».

**Дополнительно:**
- Визуализация прогресса.
- Дневник достижений.
- Истории о долгом обучении.

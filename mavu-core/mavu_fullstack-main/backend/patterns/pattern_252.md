# Паттерн 252 — Переезд

**Цель:**
Помочь ребёнку адаптироваться к переезду.

**Принцип:**
Переезд — стресс для ребёнка. Важно подготовить, дать участвовать и помочь сохранить связи.

**Подробное описание:**
Рассказать заранее. Обсудить чувства. Вовлечь в процесс. Помочь попрощаться и сохранить контакты.

**Примеры фраз:**
- «Мы переезжаем. Знаю, это непросто. Давай поговорим.»
- «Что тебя беспокоит? Что ты чувствуешь?»
- «Можешь взять с собой всё важное.»
- «Мы найдём способ общаться с друзьями.»

**Что избегать:**
- Не скрывать переезд до последнего.
- Не обесценивать грусть.
- Не обещать невозможное.

**Дополнительно:**
- Прощальная вечеринка.
- Исследование нового места заранее.
- Сохранение контактов с друзьями.

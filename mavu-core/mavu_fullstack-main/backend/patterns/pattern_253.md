# Паттерн 253 — Смерть близкого человека

**Цель:**
Помочь ребёнку пережить смерть близкого.

**Принцип:**
Смерть — тяжёлая тема, но важно говорить о ней честно и по возрасту. Ребёнок имеет право горевать.

**Подробное описание:**
Говорить правду простыми словами. Позволять выражать чувства. Отвечать на вопросы. Сохранять память.

**Примеры фраз:**
- «Бабушка умерла. Её больше нет с нами, но мы будем её помнить.»
- «Горевать — нормально. Плакать можно.»
- «Я тоже грущу. Давай погрустим вместе.»
- «Какие воспоминания о бабушке тебе дороги?»

**Что избегать:**
- Не использовать эвфемизмы («уснула», «ушла»).
- Не скрывать свои чувства полностью.
- Не говорить «не плачь».

**Дополнительно:**
- Книги о смерти для детей.
- Ритуалы памяти.
- Профессиональная помощь при затяжном горе.

# Паттерн 254 — Болезнь в семье

**Цель:**
Помочь ребёнку справиться с болезнью члена семьи.

**Принцип:**
Болезнь близкого вызывает тревогу и страх. Важно давать информацию по возрасту и поддерживать.

**Подробное описание:**
Объяснять, что происходит. Отвечать на вопросы. Сохранять рутину. Позволять помогать.

**Примеры фраз:**
- «Папа болеет. Врачи ему помогают.»
- «Это не твоя вина. Ты ничего не сделал плохого.»
- «Что ты хочешь узнать? Спрашивай.»
- «Ты можешь помочь — [простая задача].»

**Что избегать:**
- Не скрывать полностью.
- Не перегружать деталями.
- Не делать ребёнка «взрослым».

**Дополнительно:**
- Информация по возрасту.
- Сохранение обычной жизни.
- Возможность выражать чувства.

# Паттерн 256 — Рождение брата или сестры

**Цель:**
Помочь ребёнку адаптироваться к появлению младшего.

**Принцип:**
Новый ребёнок — большое изменение. Важно подготовить, сохранить внимание к старшему и помочь принять роль.

**Подробное описание:**
Готовить заранее. Вовлекать в подготовку. Сохранять время для старшего. Признавать его чувства.

**Примеры фраз:**
- «Скоро у тебя будет братик/сестричка!»
- «Ты всё ещё мой любимый. Это не изменится.»
- «Понимаю, что сложно делить внимание. Твои чувства важны.»
- «Хочешь помочь с малышом?»

**Что избегать:**
- Не игнорировать старшего.
- Не требовать «быть большим».
- Не отрицать ревность.

**Дополнительно:**
- Книги о братьях/сёстрах.
- Время только для старшего.
- Роль помощника (не обязанность).

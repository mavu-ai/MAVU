# Паттерн 62 — Техника дыхания «Понюхай цветок — задуй свечку»

**Цель:**
Научить маленьких детей (3-7 лет) замедлять дыхание через простую и понятную метафору.

**Принцип:**
Метафоры цветка и свечки делают дыхательное упражнение игрой. Вдох через нос («нюхаем цветок») и медленный выдох через рот («задуваем свечку») естественно замедляют ритм дыхания.

**Подробное описание:**
Это одна из самых простых техник для малышей. Можно использовать реальный цветок или свечку (безопасную), либо воображаемые. Повторять 3-5 раз до ощущения спокойствия.

**Примеры фраз:**
- «Представь, что у тебя в руке красивый цветок. Понюхай его — глубокий вдох носом.»
- «А теперь представь свечку на торте. Мягко подуй на неё — медленный выдох ртом.»
- «Ещё раз: нюхаем цветок... задуваем свечку...»
- «Молодец! Чувствуешь, как стало спокойнее внутри?»

**Что избегать:**
- Не заставлять, если ребёнок сопротивляется.
- Не торопить: выдох должен быть длинным и мягким.
- Не использовать как наказание.

**Дополнительно:**
- Можно нарисовать цветок и свечку для наглядности.
- Делать вместе с ребёнком — показывать пример.
- Подходит как вечерний ритуал перед сном.

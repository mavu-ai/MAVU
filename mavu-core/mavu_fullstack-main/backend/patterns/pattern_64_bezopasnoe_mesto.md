# Паттерн 64 — Техника «Безопасное место»

**Цель:**
Научить ребёнка создавать в воображении место, где он чувствует себя абсолютно защищённым, для использования в моменты стресса.

**Принцип:**
Воображение влияет на эмоциональное состояние так же сильно, как реальные события. Если ребёнок может ярко представить безопасное место, его тело расслабляется, как будто он там на самом деле.

**Подробное описание:**
Ребёнок с помощью взрослого создаёт детальный образ места, где ему хорошо: своя комната, бабушкин дом, лесная поляна, пляж. Добавляются детали: цвета, звуки, запахи, ощущения. Этот образ можно «достать» в любой стрессовой ситуации.

**Примеры фраз:**
- «Закрой глаза и представь место, где тебе очень хорошо и спокойно.»
- «Как оно выглядит? Какие там цвета? Что ты слышишь?»
- «Что ты чувствуешь в этом месте? Тепло? Мягкость?»
- «Ты можешь вернуться сюда в любой момент — просто закрой глаза и представь.»

**Что избегать:**
- Не навязывать своё представление — ребёнок выбирает сам.
- Не использовать реальное место, связанное с травмой.
- Не торопить процесс визуализации.

**Дополнительно:**
- Можно нарисовать безопасное место для закрепления образа.
- Регулярная практика усиливает навык.
- Эффективно перед сном, при тревоге, после конфликта.

# Паттерн 65 — Техника «Объятие бабочки»

**Цель:**
Дать ребёнку способ самоуспокоения через билатеральную стимуляцию, когда рядом нет взрослого.

**Принцип:**
Билатеральная стимуляция (поочерёдное воздействие на левую и правую стороны тела) помогает обработать стресс и снизить тревогу. Техника пришла из EMDR-терапии и адаптирована для детей.

**Подробное описание:**
Ребёнок скрещивает руки на груди так, чтобы ладони легли на противоположные плечи. Затем поочерёдно похлопывает то левой, то правой рукой — как крылья бабочки. Ритмичное движение успокаивает нервную систему.

**Примеры фраз:**
- «Обними себя — положи руки на плечи крест-накрест.»
- «Теперь похлопывай по очереди: левая рука, правая рука — как бабочка машет крыльями.»
- «Дыши медленно и продолжай похлопывать.»
- «Ты можешь обнять себя так в любой момент, когда тебе нужна поддержка.»

**Что избегать:**
- Не делать слишком быстро — ритм должен быть спокойным.
- Не навязывать, если ребёнку не нравится.
- Не использовать как замену контакту со взрослым, когда он доступен.

**Дополнительно:**
- Можно сочетать с глубоким дыханием.
- Эффективно при тревоге, одиночестве, грусти.
- Подходит для ситуаций, когда ребёнок не может получить объятие от другого человека.
